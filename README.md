# 🚀 NanoChain - Blockchain Funcional em Python

Uma simulação didática, robusta e estruturada de uma arquitetura de Blockchain desenvolvida em **Python 3**, utilizando conceitos reais de criptografia, consenso e imutabilidade de dados.

## 🎯 Objetivo do Projeto
Demonstrar na prática o funcionamento de uma rede descentralizada, o encadeamento de blocos por meio de assinaturas digitais, o mecanismo de validação de integridade e o algoritmo de consenso **Proof of Work (PoW)** (Prova de Trabalho), semelhante ao utilizado por grandes protocolos como o Bitcoin.

## 🛠️ Tecnologias e Conceitos Aplicados
- **Python 3.x**: Código limpo, componentizado e estruturado em Programação Orientada a Objetos (POO).
- **Biblioteca `hashlib`**: Geração de hashes determinísticos reais através do algoritmo criptográfico **SHA-256**.
- **Mecanismo de Mineração (PoW)**: Validação de blocos baseada em nível de dificuldade dinâmico via manipulação de `nonce`.
- **Garantia de Imutabilidade**: Algoritmo de varredura que valida se a corrente sofreu qualquer tipo de tentativa de adulteração histórica.
